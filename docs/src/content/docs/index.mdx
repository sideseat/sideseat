## Quick Start

Get up and running with SideSeat in minutes:

```bash
# Clone the repository
git clone https://github.com/spugachev/sideseat.git
cd sideseat

# Install dependencies and setup
make setup

# Start development servers
make dev
```

Open [http://localhost:5173](http://localhost:5173) to access the dashboard.

## Features

<CardGrid stagger>
  <Card title="OpenTelemetry Collector" icon="magnifier">
    Built-in OTLP-compatible trace collector with automatic AI framework detection and GenAI field extraction.
  </Card>
  <Card title="Real-time Streaming" icon="rocket">
    SSE-based live trace updates for monitoring agent execution as it happens.
  </Card>
  <Card title="Framework Detection" icon="puzzle">
    Automatic detection of LangChain, LlamaIndex, Strands, and other AI frameworks with normalized field extraction.
  </Card>
  <Card title="Efficient Storage" icon="laptop">
    Parquet-based storage with FIFO retention, bounded memory buffers, and disk safety monitoring.
  </Card>
</CardGrid>

## Tech Stack

- **Backend**: Rust with Axum framework
- **Frontend**: React 19 with Vite
- **Telemetry**: OpenTelemetry (OTLP) with Parquet storage
- **Database**: SQLite with migrations
- **CLI**: Distributed as NPM package
- **Docs**: Astro Starlight

## Requirements

- Rust 1.75 or higher
- Node.js 20.19+ or 22.12+
- Make
