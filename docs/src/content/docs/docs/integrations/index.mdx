SideSeat collects runs from any application emitting OpenTelemetry data. Most AI frameworks have built-in OpenTelemetry support or can be instrumented with the SideSeat SDKs.

## How Integration Works

1. **Install SDK**: Add the SideSeat Python or JavaScript SDK
2. **Initialize**: Call `SideSeat()` or `sideseat_init()`
3. **Run**: Your framework's telemetry flows to the local workbench automatically

SideSeat then:
- Auto-detects the framework
- Extracts model, token, and cost information
- Normalizes messages into a unified run timeline

<Aside type="tip">
Not sure which path to use? See [Instrumentation Options](/docs/instrumentation/).
</Aside>
