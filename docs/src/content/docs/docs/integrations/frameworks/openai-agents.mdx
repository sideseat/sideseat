[OpenAI Agents SDK](https://github.com/openai/openai-agents-python) is OpenAI's framework for building AI agents. SideSeat captures full agent-level traces including tool calls, handoffs, and LLM calls.

## Prerequisites

- SideSeat running locally (`sideseat`)
- Python 3.9+
- OpenAI API credentials configured

## Quick Start

<Steps>
1. **Start SideSeat**

   ```bash
   npx sideseat
   ```

2. **Install dependencies**

   <Tabs>
     <TabItem label="pip">
       ```bash
       pip install openai-agents "sideseat[openai-agents]"
       ```
     </TabItem>
     <TabItem label="uv">
       ```bash
       uv add openai-agents "sideseat[openai-agents]"
       ```
     </TabItem>
   </Tabs>

3. **Add telemetry**

   ```python
   from sideseat import SideSeat, Frameworks
   from agents import Agent, Runner

   SideSeat(framework=Frameworks.OpenAIAgents)

   agent = Agent(
       name="Assistant",
       instructions="You are a helpful assistant."
   )

   result = Runner.run_sync(agent, "Hello!")
   print(result.final_output)
   ```

4. **View runs**

   Open [http://localhost:5388](http://localhost:5388) to see your runs.
</Steps>

## Without SideSeat SDK

Use `logfire` to instrument the OpenAI Agents SDK via its official hook:

<Steps>
1. **Start SideSeat**

   ```bash
   npx sideseat
   ```

2. **Set the endpoint**

   ```bash
   export OTEL_EXPORTER_OTLP_ENDPOINT=http://localhost:5388/otel/default
   ```

3. **Install dependencies**

   <Tabs>
     <TabItem label="pip">
       ```bash
       pip install openai-agents "logfire>=4.29.0" opentelemetry-exporter-otlp
       ```
     </TabItem>
     <TabItem label="uv">
       ```bash
       uv add openai-agents "logfire>=4.29.0" opentelemetry-exporter-otlp
       ```
     </TabItem>
   </Tabs>

4. **Add telemetry**

   ```python
   import logfire
   from opentelemetry import trace
   from opentelemetry.sdk.trace.export import BatchSpanProcessor
   from opentelemetry.exporter.otlp.proto.http.trace_exporter import OTLPSpanExporter

   logfire.configure(send_to_logfire=False, console=False)
   logfire.instrument_openai_agents()

   provider = trace.get_tracer_provider()
   provider.add_span_processor(BatchSpanProcessor(OTLPSpanExporter()))
   ```

5. **View runs**

   Open [http://localhost:5388](http://localhost:5388) to see your runs.
</Steps>

## Tools

Define tools using the `@function_tool` decorator:

```python
from agents import Agent, Runner, function_tool

@function_tool
def get_weather(city: str) -> str:
    """Get weather for a city."""
    return f"Sunny in {city}"

agent = Agent(
    name="WeatherAssistant",
    instructions="Use get_weather to provide forecasts.",
    tools=[get_weather]
)

result = Runner.run_sync(agent, "What's the weather in Paris?")
```

SideSeat shows:
- Parent span for the agent run
- Child spans for each OpenAI call
- Child spans for each tool execution

## Session Tracking

Add session context with manual spans:

```python
from opentelemetry import trace

tracer = trace.get_tracer(__name__)

with tracer.start_as_current_span(
    "openai_agents.session",
    attributes={
        "session.id": "session-123",
        "user.id": "user-456"
    }
):
    result = Runner.run_sync(agent, "Hello!")
```

## Extracted Attributes

| Attribute | Description |
|-----------|-------------|
| `gen_ai.system` | `openai` |
| `gen_ai.request.model` | OpenAI model ID |
| `gen_ai.usage.input_tokens` | Input token count |
| `gen_ai.usage.output_tokens` | Output token count |
| `logfire.span_type` | Span type (present when using SideSeat SDK) |

## Async Usage

The SDK supports async execution:

```python
import asyncio
from agents import Agent, Runner

agent = Agent(
    name="Assistant",
    instructions="You are a helpful assistant."
)

async def main():
    result = await Runner.run(agent, "Hello!")
    print(result.final_output)

asyncio.run(main())
```
