[Strands Agents](https://strandsagents.com) is an AI agent framework developed by AWS. It has native OpenTelemetry support, making SideSeat integration seamless.

## Quick Start

<Steps>
1. **Start SideSeat**

   ```bash
   npx sideseat
   ```

2. **Install dependencies**

   <Tabs>
     <TabItem label="pip">
       ```bash
       pip install strands-agents sideseat
       ```
     </TabItem>
     <TabItem label="uv">
       ```bash
       uv add strands-agents sideseat
       ```
     </TabItem>
   </Tabs>

3. **Add telemetry**

   ```python
   from sideseat import SideSeat, Frameworks
   from strands import Agent

   SideSeat(framework=Frameworks.Strands)

   agent = Agent()
   response = agent("What is the capital of France?")
   print(response)
   ```

4. **View runs**

   Open [http://localhost:5388](http://localhost:5388) to see your runs.
</Steps>

## Without SideSeat SDK

<Steps>
1. **Start SideSeat**

   ```bash
   npx sideseat
   ```

2. **Set the endpoint**

   ```bash
   export OTEL_EXPORTER_OTLP_ENDPOINT=http://localhost:5388/otel/default
   ```

3. **Install dependencies**

   <Tabs>
     <TabItem label="pip">
       ```bash
       pip install 'strands-agents[otel]'
       ```
     </TabItem>
     <TabItem label="uv">
       ```bash
       uv add 'strands-agents[otel]'
       ```
     </TabItem>
   </Tabs>

4. **Add telemetry**

   ```python
   from strands.telemetry import StrandsTelemetry
   from strands import Agent

   telemetry = StrandsTelemetry()
   telemetry.setup_otlp_exporter()
   telemetry.setup_meter(enable_otlp_exporter=True)

   agent = Agent()
   response = agent("What is the capital of France?")
   print(response)
   ```

5. **View runs**

   Open [http://localhost:5388](http://localhost:5388) to see your runs.
</Steps>

<Aside type="note">
The SideSeat SDK includes additional features like binary data handling and framework auto-detection. Use it unless you need Strands-specific telemetry configuration.
</Aside>

## What You'll See

SideSeat shows a trace timeline with:
- A parent span for the agent run
- Child spans for each LLM call with model, tokens, and cost
- Child spans for each tool execution with inputs and outputs

## Tools

Strands automatically traces tool calls. Define tools normally:

```python
from strands import Agent, tool

@tool
def get_weather(location: str) -> str:
    """Get weather for a location."""
    return f"Weather in {location}: Sunny, 72F"

agent = Agent(tools=[get_weather])
response = agent("What's the weather in Paris?")
```

## Sessions

Group related traces using trace attributes:

```python
agent = Agent(
    trace_attributes={
        "session.id": "conversation-123",
        "user.id": "user-456",
    }
)
```

These attributes appear in SideSeat and enable filtering. All turns with the same `session.id` are grouped under one session.
