## Option 1: SideSeat SDK (Recommended)

### Python

```python
from sideseat import SideSeatTelemetry
SideSeatTelemetry().setup_otlp_exporter()
```

### JavaScript

```typescript
import { sideseat_init } from '@sideseat/sdk';
sideseat_init();
```

## Option 2: Framework Native OTLP

Many frameworks emit OpenTelemetry out of the box. Point them to SideSeat:

```text
http://localhost:5388/otel/default/v1/traces
```

Start with Strands or LangGraph: [Integrations](/docs/integrations/).

## Option 3: Generic OpenTelemetry

```python
from opentelemetry import trace
from opentelemetry.sdk.trace import TracerProvider
from opentelemetry.sdk.trace.export import BatchSpanProcessor
from opentelemetry.exporter.otlp.proto.http.trace_exporter import OTLPSpanExporter

provider = TracerProvider()
provider.add_span_processor(
    BatchSpanProcessor(
        OTLPSpanExporter(endpoint="http://localhost:5388/otel/default/v1/traces")
    )
)
trace.set_tracer_provider(provider)
```
