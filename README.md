# SideSeat

Your unified workspace for building and debugging AI agents.

## Quick Start

```bash
make setup
make dev
```

Open http://localhost:5173 for the dashboard

## Requirements

- Rust 1.75+
- Node.js 20.19+ or 22.12+
- Make
