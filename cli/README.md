# @sideseat/cli

AI development toolkit - proxy, observability, debugging, and optimization for AI applications.

## Installation

```bash
npm install -g @sideseat/cli
```

## Usage

```bash
sideseat start              # Start all services
sideseat --help             # Show help
```

## Features

- AI Proxy (OpenAI, Anthropic, etc.)
- OpenTelemetry Collector
- MCP & A2A Protocol Support
- Prompt Monitoring & Optimization
- Unified Dashboard

## Requirements

- Node.js 20.19+ or 22.12+

## Supported Platforms

- macOS (Intel & Apple Silicon)
- Linux (x64 & ARM64)
- Windows (x64)
